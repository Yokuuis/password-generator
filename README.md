# 🔐 password generator

Generador de contraseñas seguras programada en python

## 📥 Instalación

Para obtener una copia local de este proyecto, sigue estos pasos:

### 1. Clonar el repositorio

```bash
git clone https://github.com/Yokuuis/password-generator.git
```

### 2. Acceder a la carpeta del proyecto

```bash
cd password-generator
```

---

## 🚀 Uso

Una vez dentro del directorio, ejecuta el programa principal:

```bash
python main.py
```

> **Nota:** Asegúrate de tener instalado Python en su version mayor actualizada,  si el archivo principal tiene otro nombre, sustituye `main.py` por el correspondiente.

---

## 🛠️ Características

- ✅ **Seguridad:** Generación de contraseñas aleatorias de alta entropía.
- ✅ **Ligero:** Utiliza únicamente librerías estándar de Python.
- ✅ **Fácil de usar:** Interfaz sencilla y rápida de ejecutar.
